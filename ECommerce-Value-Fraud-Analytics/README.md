# 📊 E-Commerce Customer Value & Satisfaction Analytics

An End-to-End Business Intelligence and Machine Learning project that analyzes customer behavior using SQL, builds an RFM (Recency, Frequency, Monetary) Segmentation model, and predicts customer satisfaction using a Random Forest Classifier.

## 🛠️ Features & Pipeline
1. **Data Infrastructure:** Cleaned and loaded raw CSV data into a **SQLite Database** using Python.
2. **Advanced Customer Analytics:** Performed **RFM Segmentation** to classify customers into actionable business segments (Champions, Loyal, At Risk, Hibernating).
3. **Machine Learning Predictive Modeling:** Developed a **Random Forest Classifier** with **98.57% Accuracy** to predict customer satisfaction levels.
4. **Interactive BI Dashboard:** Built a dynamic dashboard using **Streamlit** with real-time business KPIs and interactive charts.

## 📁 Project Structure
- `data/`: Contains the raw e-commerce dataset.
- `db_setup.py`: Script to initialize and populate the SQLite database.
- `notebook.ipynb`: Jupyter notebook for EDA, RFM analysis, and ML modeling.
- `app.py`: Streamlit dashboard application code.
- `requirements.txt`: List of required Python packages.
